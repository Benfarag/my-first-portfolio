# 🌐 Benfarag Visual Portfolio

مرحباً بك في مستودع الكود الخاص بموقعي الشخصي. هذا الموقع يعكس هويتي كمطور مهتم بتبسيط الواجهات ودمج الفن الرقمي بالبرمجة.

## 🚀 نظرة عامة (Overview)
تصميم عصري، نظيف، ومتجاوب بالكامل يعرض مهاراتي ومشاريعي البرمجية. تم التركيز في هذا المشروع على تجربة المستخدم (UX) واستخدام تقنيات الـ CSS الحديثة.

## 🛠 التقنيات المستخدمة (Tech Stack)
- **Frontend:** HTML5, CSS3 (Flexbox & Grid), JavaScript
- **Icons:** Font Awesome & Google Fonts
- **Graphics:** 3D Renders (Blender) & Pixel Art
- **Environment:** Developed on Fedora Linux using VS Code

## ✨ المميزات (Features)
- **Minimalist Design:** واجهة هادئة تركز على المحتوى.
- **Responsive Layout:** متوافق مع كافة أحجام الشاشات (Mobile, Tablet, Desktop).
- **Interactive Elements:** تأثيرات Hover وحركات تفاعلية للأيقونات والأزرار.
- **Fast Loading:** تحسين الصور والأكواد لضمان سرعة التحميل.
